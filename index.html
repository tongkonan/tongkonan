how to buy ?

Step 1 - Create MetaMask/Trust wallet

Create a MetaMask Wallet using either a desktop computer or an TrustWallet using Android Mobile Device. That will allow you to send, and receive $TKN

Step 3 - Connect your wallet to Cryptopreneur
Access your wallet to Cryptopreneur.Finance by clicking ‘Connect Wallet’ and selecting MetaMask if You Use Desktop & Trust Wallet if You Use Android Mobile Device.
Step 2 - Send BNB to your wallet

You can transfer it to your MetaMask/Trust Wallet from exchanges like Pintu, Indodax, TokoCrypto, Binance, etc. Make sure to use the BEP-20 Network when transferring BNB.

Step 4 - Swap BUSD for $TKN

You can start swapping as soon as you have BUSD available! Press ‘Select a currency’ and enter the $TKN token address.
